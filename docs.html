معرفی APAL

APAL (Afghan Programming Language) یک زبان برنامه‌نویسی مدرن، ساده و قدرتمند است که با هدف حذف موانع ورود به دنیای تکنولوژی برای فارسی‌زبانان و پشتو‌زبانان طراحی شده است. ما معتقدیم که زبان برنامه‌نویسی باید طبیعی و خوانا باشد، نه یک کد رمزگذاری شده.

کدهای نوشته شده با APAL توسط موتور ترجمه قدرتمند ما به جاوااسکریپت تبدیل می‌شوند تا در تمام مرورگرهای مدرن وب قابل اجرا باشند. این یعنی شما با یادگیری APAL، در حال یادگیری یکی از محبوب‌ترین زبان‌های دنیا نیز هستید.

نوشتن اولین کد

شروع کار با APAL به سادگی نوشتن یک جمله است. این کد، یک پیام ساده را در کنسول مرورگر شما چاپ می‌کند:

چاپ کن به "سلام، دنیا! من از APAL استفاده می‌کنم."

کد جاوااسکریپت تولید شده توسط موتور APAL:

console.log("سلام، دنیا! من از APAL استفاده می‌کنم.");

متغیرها

برای ذخیره اطلاعات در برنامه، از کلمه‌ی کلیدی ذخیره کن استفاده می‌کنیم. برای اختصاص دادن یک مقدار، از کلمه‌ی مساوی بهره می‌بریم.

// ذخیره یک متن
ذخیره کن نام کاربر مساوی "سارا"

// ذخیره یک عدد
ذخیره کن سن کاربر مساوی ۲۵

// ذخیره نتیجه یک محاسبه
ذخیره کن سن بعدی مساوی سن کاربر جمع ۱

کد جاوااسکریپت تولید شده:

let نام_کاربر = "سارا";
let سن_کاربر = 25;
let سن_بعدی = سن_کاربر + 1;

منطق و شرط

برای تصمیم‌گیری در برنامه، از ساختار اگر و گرنه استفاده می‌کنیم. بلوک‌های کد با استفاده از تورفتگی (۴ فاصله) مشخص می‌شوند.

اگر سن کاربر بزرگتر از ۱۸
    چاپ کن به "کاربر بزرگسال است و می‌تواند وارد شود."
گرنه
    چاپ کن به "دسترسی برای کاربران زیر ۱۸ سال مجاز نیست."

کد جاوااسکریپت تولید شده:

if (سن_کاربر > 18) {
    console.log("کاربر بزرگسال است و می‌تواند وارد شود.");
} else {
    console.log("دسترسی برای کاربران زیر ۱۸ سال مجاز نیست.");
}

حلقه‌های تکرار

برای انجام یک کار به صورت مکرر، از حلقه‌های تکرار کن روی استفاده می‌کنیم. این حلقه برای شمارش اعداد یا پیمایش یک لیست ایده‌آل است.

تکرار کن روی هر شماره از ۱ تا ۵
    چاپ کن به "شماره:" و شماره

کد جاوااسکریپت تولید شده:

for (let شماره = 1; شماره <= 5; شماره++) {
    console.log("شماره:" + شماره);
}

توابع

برای سازماندهی بهتر کد و جلوگیری از تکرار، می‌توانید توابع خود را با کلمه‌ی کلیدی کار تعریف کنید. توابع به شما اجازه می‌دهند یک قطعه کد را نام‌گذاری کرده و آن را هر زمان که خواستید، فراخوانی کنید.

// تعریف یک تابع برای محاسبه مجموع دو عدد
کار مجموع را محاسبه کن با عدد اول و عدد دوم
    ذخیره کن نتیجه مساوی عدد اول جمع عدد دوم
    برگردان نتیجه

// فراخوانی تابع و ذخیره نتیجه آن
ذخیره کن جمع نهایی مساوی مجموع را محاسبه کن با ۱۰ و ۲۰
چاپ کن به "مجموع نهایی برابر است با:" و جمع نهایی

کد جاوااسکریپت تولید شده:

function مجموع_را_محاسبه_کن(عدد_اول, عدد_دوم) {
    let نتیجه = عدد_اول + عدد_دوم;
    return نتیجه;
}

let جمع_نهایی = مجموع_را_محاسبه_کن(10, 20);
console.log("مجموع نهایی برابر است با:" + جمع_نهایی);

مثال پیشرفته: کار با وب

قدرت واقعی APAL در سادگی تعامل با عناصر وب است. این مثال نشان می‌دهد چگونه می‌توانید با یک کلیک روی یک دکمه، متن یک عنوان را تغییر دهید.

فرض کنید این کد HTML را دارید:

<h1 id="main-title">سلام دنیا</h1>
<button id="change-title-btn">عنوان را تغییر بده</button>

کد APAL برای مدیریت این عناصر:

کار عنوان را تغییر بده
    تغییر بده متن بگیر با آیدی "main-title" مساوی "سلام از APAL!"

وقتی کلیک شد روی بگیر با آیدی "change-title-btn"
    عنوان را تغییر بده

کد جاوااسکریپت تولید شده:

function عنوان_را_تغییر_بده() {
    document.getElementById("main-title").innerText = "سلام از APAL!";
}

document.getElementById("change-title-btn").addEventListener("click", عنوان_را_تغییر_بده);
